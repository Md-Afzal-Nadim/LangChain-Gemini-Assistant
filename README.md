## 📌 Project Overview

This project is a full-stack AI search assistant that integrates Large Language Models (LLMs) with real-time internet search. Users can sign up, verify their email, chat with AI, access web-powered responses, and view their complete chat history.

## ✨ Key Features

* Secure User Authentication
* User Registration & Login
* Email Verification using Nodemailer
* Real-Time AI Chat
* Real-Time Communication with Socket.io
* AI-Powered Responses
* Internet Search Integration
* Persistent Chat History
* MongoDB Database Storage
* Responsive UI for Desktop and Mobile
* Full-Stack Architecture
